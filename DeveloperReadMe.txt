Developer Read Me:

The important files are main.js and index.html

The index.html file contains the templates for each page (view) of the app
This is a single view application which uses backbone, underscore, and jquery mobile
Hammer js is used for swiping and highcharts manages the chart view
Time circles manages the animations on the feed screen

In the main.js file first all the views are listed, then the routes are established, and finally each view's javascript is defined. The hash is changed whenever the view changes.
